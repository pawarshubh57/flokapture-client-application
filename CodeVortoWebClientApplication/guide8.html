I want to learn how entities are being used in my project.

How to interpret and use this tool?

  • From the left Reports menu I will take you to the CRUD Activity report which displays in the body of the page.



  • The CRUD Activity report displays results in a grid with the following details:



    • Entity Name

    • Create

    • Read

    • Update

    • Delete

    • Object Name

    • Object Type

  • The X icon at the top of the page allows you to export the results to an Excel spreadsheet.

  • The Search text box next to the X icon at the top of the page can be used to filter results and will filter on Entity Name and Object Name.

  • Each of the column headers can be used to sort the column ascending/descending.
